## 🚀 Featured Projects

### 🔐 [Neokey : Zero-knowledge Vaultless Password Platform](https://github.com/Rushikesh-Yeole/Neokey)  
✨ A vaultless password synthesis platform that derives passwords deterministicaly on client-side at runtime. Eliminates 100% of persistent PII/secrets and renders data exfiltration useless.

⚙️ **Stack:** Node.js · React · MongoDB · Redis · Argon2id · BLAKE3 · GCP <br>
🔗 [Live Beta in production](https://neokey.in)

### 🌐 [Horizon : Career Intelligence Platform](https://github.com/Rushikesh-Yeole/Horizon)
✨ A self-improving career intelligence platform that gives users cited roadmaps and honest fit scores synthesized from live JDs, real hiring signals, and a knowledge graph that gets sharper with every user.

⚙️ **Stack:** Gemini LLM · FastAPI · Neo4j · MongoDB · Redis · Tavily
